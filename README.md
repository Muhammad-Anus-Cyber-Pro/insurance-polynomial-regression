# 📊 Regression Analysis on Insurance Dataset

## 🚀 Project Overview

In this project, I applied **Linear Regression** and **Polynomial Regression** on an Insurance dataset to evaluate their performance on real-world data.

The aim was to understand how well these models generalize and handle complex patterns.

---

## 🧠 What I Did

* Applied **Linear Regression** as a baseline
* Applied **Polynomial Regression** to capture non-linear relationships
* Evaluated both models using:

  * R² Score
  * Mean Squared Error (MSE)
* Visualized predictions vs actual values

---

## 📉 Results & Observations

* Linear Regression showed **poor performance**
* Polynomial Regression also **failed to significantly improve results**
* Increasing polynomial degree led to:

  * Overfitting in some cases
  * No meaningful generalization

---

## 💡 Key Learnings

* Real-world datasets often violate assumptions of simple regression models
* Polynomial models are not always the solution for non-linearity
* Overfitting is a real challenge when increasing model complexity
* Understanding model failure is crucial for improvement
